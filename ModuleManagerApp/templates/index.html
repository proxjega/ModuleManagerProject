Module Manager

A Django web application for managing subject modules. Authenticated users can create, update, and view their own module data and generate PDF documents from module information.

Login Register

Architecture

The project follows Django's Model-View-Template pattern to keep data models, request handling, and UI rendering clearly separated.

Design Principles

  • Models handle persistence and database mapping.
  • Forms validate user input.
  • Views handle request routing and responses.
  • Services contain business logic like registration and module management.

Features

  • User authentication
  • Per-user module management
  • Module creation and editing
  • Module information export to PDF
